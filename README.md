# Offensive Security Roadmap

A structured, beginner-friendly path into offensive security and penetration testing — built for AEC Coding Club members who want to go from zero to their first CTF, box, or bug bounty.

## Why this exists

Most coding clubs cover web dev, app dev, ML, competitive programming — but cybersecurity, especially the *offensive* side, rarely gets a proper introduction. This repo is my attempt to fix that: a curated roadmap, cheat sheets, and writeups that lower the barrier to entry for anyone in the club curious about ethical hacking.

This isn't a finished product — it's a living document I'll keep expanding as I learn and as I (hopefully) get to run sessions on this as part of Core Committee.

## About me

I'm currently working towards the **CPTS (Certified Penetration Testing Specialist)** certification from HackTheBox, focused on offensive security — web exploitation, Active Directory attacks, and privilege escalation. This repo documents both my own learning and resources I wish existed when I started.

## Who this is for

Complete beginners with basic Linux/networking curiosity, all the way up to people ready to start their first CTF or certification path. No prior security experience assumed.
